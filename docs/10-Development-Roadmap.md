# CipherWolf Development Roadmap

## Phase 1: Backend Foundation

- Install backend requirements.
- Finalize PostgreSQL and Redis services.
- Create database schema.
- Add secure admin login.
- Keep frontend API paths stable.

Status: foundation created.

## Phase 2: Login

- Connect login UI to backend.
- Protect admin routes.
- Add logout/session expiry behavior.
- Add forgot-password flow later after email service is selected.

## Phase 3: Admin Dashboard

- Show live totals.
- Show recent contacts and events.
- Add realtime WebSocket updates.
- Add dashboard layout for later SOC and CRM modules.

## Phase 4: Visitors

- Track page views.
- Add unique visitor ID.
- Add device/browser/location fields.
- Build visitor analytics charts.

## Phase 5: SOC Dashboard

- Add threat events.
- Add severity, source, IP, status, and timeline.
- Add realtime alerts.
- Add filters and export.

## Phase 6: CMS

- Blog posts.
- Projects.
- Services.
- Public site content editing.

## Phase 7: CRM

- Leads.
- Pipeline status.
- Notes.
- Follow-up tasks.
- Contact history.
